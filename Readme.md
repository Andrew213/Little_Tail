# [LittleTail](https://effulgent-praline-98eb7c.netlify.app/)
Простой fullstack пет проект ветеринарной клиники. 

С возможностями:
- Регистрация пользователя
- Авторизация через jwt токен
- Добавление питомца
- Поиска питомка
- Удаление питомца
- Пагинация
- Запись питомца к врачу на выбранную дату
